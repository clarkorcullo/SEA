# 🛡️ Social Engineering Awareness Program (SEAP)

**A comprehensive educational platform for cybersecurity awareness and social engineering defense**

SEAP is a professional, enterprise-grade learning web application that helps students and staff identify social engineering attacks, respond effectively during security incidents, and build robust online security habits. Built with Flask and featuring responsive design, professional certificate generation, and cross-device compatibility.

## 🌐 **Live Application**
- **Production URL**: `https://mmdcsea.onrender.com`
- **Repository**: `https://github.com/clarkorcullo/SEA`
- **Version**: 2.0.0 (Enhanced Cross-Device Compatibility)
- **Institution**: Mapúa Malayan Digital College (MMDC)
- **Security Rating**: A- (Excellent) - [View Security Audit](SECURITY_AUDIT_REPORT.md)

## 📚 **Curriculum Overview**

### **Module Structure**
1. **Module 1**: Social Engineering Fundamentals - Understanding the basics
2. **Module 2**: Phishing & Digital Scams - Recognizing and preventing phishing attacks
3. **Module 3**: Proactive Digital Defense - Building strong security habits
4. **Module 4**: Incident Response & Reporting - Immediate action during security incidents
5. **Module 5**: Advanced Threats & Continuous Awareness - Evolving threat landscape
6. **Final Assessment**: Comprehensive 25-question evaluation

### **Learning Components**
Each module includes:
- **📖 Educational Content**: Videos, images, and comprehensive lessons
- **💭 Reflection System**: Submit and view peer reflections
- **📝 Knowledge Checks**: 5-question assessments with 80% passing threshold
- **🎮 Interactive Simulations**: Real-world scenario-based learning (Modules 2-5)
- **📊 Progress Tracking**: Detailed progress monitoring and analytics

## 🔧 **Technical Specifications**

### **Technology Stack**
- **Backend**: Flask 3.0.0 with SQLAlchemy ORM
- **Database**: PostgreSQL (production) / SQLite (development)
- **Frontend**: Bootstrap 5 with responsive design
- **Authentication**: Flask-Login with secure sessions
- **Security**: Enterprise-grade security with A- rating
- **Deployment**: Render.com with Gunicorn WSGI server

### **System Requirements**
- **Python**: 3.11.9+
- **Memory**: 512MB RAM (minimum)
- **Storage**: 1GB (for development)
- **Browser**: Modern browsers (Chrome, Firefox, Safari, Edge)
- **Network**: Internet connection for video content

## 🚀 **Quick Start Guide**

### **Local Development Setup**
```bash
# 1. Clone the repository
git clone https://github.com/clarkorcullo/SEA.git
cd SEA

# 2. Install dependencies
pip install -r requirements.txt

# 3. Start the application
python app.py

# 4. Access the application
# Open browser: http://localhost:5000
```

### **Production Deployment**
- **Platform**: Render.com (automated deployment)
- **Database**: PostgreSQL for data persistence
- **Configuration**: Environment-based settings
- **Monitoring**: Health check endpoint at `/health`
